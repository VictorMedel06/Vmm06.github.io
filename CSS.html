Contenido de CSS

CSS (Cascading Style Sheets) es un lenguaje de estilo utilizado para describir la presentación de un documento escrito en HTML o XML. CSS controla el diseño, el color, las fuentes y el espaciado de los elementos en una página web.

Propiedad `color`

Se utiliza para cambiar el color del texto:

p { color: red; }

Resultado:

Este texto es rojo.

Leer más en W3C

Propiedad `background-color`

Se utiliza para cambiar el color de fondo de un elemento:

div { background-color: lightblue; }

Resultado:

Este es un div con fondo azul claro.
Leer más en W3C

Propiedad `font-size`

Se utiliza para cambiar el tamaño de la fuente:

h1 { font-size: 2em; }

Resultado:

Este es un título grande.

Leer más en W3C

Propiedad `margin`

Se utiliza para crear espacio alrededor de los elementos:

p { margin: 20px; }

Resultado:

Este párrafo tiene un margen de 20 píxeles alrededor.
