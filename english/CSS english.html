CSS Content

CSS (Cascading Style Sheets) is a style language used to describe the presentation of a document written in HTML or XML. CSS controls the layout, color, fonts, and spacing of elements on a web page.

The `color` Property

Used to change the text color:

p { color: red; }

Result:

This text is red.

Read more on W3C

The `background-color` Property

Used to change the background color of an element:

div { background-color: lightblue; }

Result:

This is a div with a light blue background.
Read more on W3C

The `font-size` Property

Used to change the size of the font:

h1 { font-size: 2em; }

Result:

This is a large heading.

Read more on W3C

The `margin` Property

Used to create space around elements:

p { margin: 20px; }

Result:

This paragraph has a 20-pixel margin around it.

Read more on W3C
