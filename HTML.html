Conceptos Básicos de HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Se utiliza para estructurar el contenido en la web de manera semántica.

Etiqueta "<p>": Párrafos

La etiqueta "<p>" se utiliza para definir párrafos en HTML. Veamos un ejemplo básico:

&lt;p&gt;Este es un párrafo en HTML.&lt;/p&gt;

Resultado:

Este es un párrafo en HTML.

Más información sobre la etiqueta en la documentación oficial de HTML: W3C HTML Documentation

Etiqueta "<a>": Enlaces

La etiqueta "<a>" es usada para crear enlaces en HTML que redirigen a otras páginas o recursos externos. Veamos su sintaxis básica:

&lt;a href="https://www.ejemplo.com"&gt;Visitar Ejemplo&lt;/a&gt;

Resultado:

Visitar Ejemplo

Etiqueta "<img>": Imágenes

La etiqueta "<img>" es utilizada para insertar imágenes en una página web. Veamos un ejemplo básico:

&lt;img src="imagen.jpg" alt="Descripción de la imagen"&gt;
